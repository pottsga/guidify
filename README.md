# Guidify

An Obsidian plugin that renames files to GUIDs within specified base directories.

## Features
- Automatically renames files to GUIDs when they are created or moved into specified base directories.
- Configurable base directories for GUID renaming.
- User-friendly settings tab for easy configuration.

## Installation

### BRAT
1. Open Obsidian and go to "Settings" > "Community Plugins".
2. Click on "Browse" and search for "BRAT".
3. Install and enable the BRAT plugin.
4. Use BRAT to install "Guidify" from this repository. 
5. Copy the https://github.com/pottsga/guidify URL and paste it into BRAT's install dialog.

### Manual Installation
1. Clone or download this repository.
2. Copy the `guidify` folder into your Obsidian vault's `.obsidian/plugins/` directory.
3. Enable the plugin in Obsidian's settings under "Community Plugins".
4. Configure the base directories in the plugin settings.

## Usage
1. After installation, go to the plugin settings to specify the base directories where you want files to be renamed to GUIDs.
2. Create or move files into the specified directories, and they will be automatically renamed to GUIDs.
